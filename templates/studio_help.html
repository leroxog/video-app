Programmiersprachen-Grundlagen

In timeskip studio kannst du dein Spiel in Python (empfohlen), HTML, C#, JavaScript oder Java programmieren — alle Fünf können genau dasselbe, nur die Syntax sieht anders aus. Wähle unten deine Sprache, um die passenden Befehle zu sehen.

Wie eine Regel aufgebaut ist

Jede Regel besteht aus diesen Teilen, in dieser Reihenfolge:

  1. REPEAT (optional) — wiederholt die Regel jeden Frame statt nur einmal.
  2. Block — für welchen Teil des Spiels die Regel gilt.
  3. Wann — berührt / geklickt / immer.
  4. Bedingung (optional) — die Regel feuert nur, wenn sie wahr ist.
  5. Aktion — was passieren soll.
  6. Ende — SOLID (fest) oder PASSABLE (durchlässig) schließt die Regel ab.

Im Editor beginnt jede Zeile automatisch mit dem Pfeil-Symbol "⇒", und ein fertiges Skript endet mit "⇓" — das musst du nicht selbst eintippen, das übernimmt der Editor.
